# Changelog

## 0.5.1 (2026-05-17)

- **`/medium`** — случайный вопрос средней сложности (★★☆)
- **`/mistakes`** — список вопросов с ошибками (отдельно от `/review`)
- **`/new <тема>`** — только непройденные вопросы по компетенции
- **`/export`**: сколько вопросов банка ещё не встречалось
- Достижения: **серия 15+**, **точность 80%+** (от 10 ответов)
- **+3 вопроса** (42 в банке)

## 0.5.0 (2026-05-17)

- **`/new`** — только вопросы, которые вы ещё не встречали
- **`/question <id>`** (и `/q`) — открыть конкретный вопрос из банка
- **Достижения по темам**: «Освоена тема: …» при прохождении всех вопросов компетенции
- **`/status`**: дневная цель и освоение банка
- **+3 вопроса** (39 в банке)

## 0.4.9 (2026-05-17)

- **Вторая попытка** на вопрос без штрафа в статистике (после первой ошибки)
- **`/giveup`** — показать ответ и завершить вопрос (засчитывается как ошибка)
- **`/skip`** — по-прежнему без штрафа; в тексте вопроса подсказки `/hint · /giveup · /skip`
- **Умный `/quiz`**: чуть выше шанс вопросов, где раньше были ошибки
- **+3 вопроса** (36 в банке)

## 0.4.8 (2026-05-17)

- **`/export`** — текстовый отчёт о прогрессе (удобно сохранить или переслать)
- **`/easy`** — случайный лёгкий вопрос; **`/mistakes`** — алиас `/review`
- **`/map`**: освоено вопросов банка по каждой теме
- После ошибки подсказка про `/review`
- **+3 вопроса** (33 в банке)

## 0.4.7 (2026-05-17)

- **Дневная цель** (`DAILY_GOAL`, по умолчанию 5) — счётчик ответов за сегодня в `/start` и `/stats`
- **`/challenge`** — случайный вопрос максимальной сложности
- **`/current`** — повторно показать активный вопрос
- **`mentor check --print-config`**: разбивка банка по сложности
- **+3 вопроса** (30 в банке)

## 0.4.6 (2026-05-17)

- **`/start`** — персональное приветствие и подсказка, что тренировать
- **`/achievements`** — достижения (серии, банк, вехи по ответам)
- **`/stats`**: освоено вопросов банка (≥1 верный) и блок достижений
- **+3 вопроса** (27 в банке): confusion matrix, SMOTE, matplotlib

## 0.4.5 (2026-05-17)

- **`/next`** — быстрый алиас следующего вопроса (как `/quiz` без аргументов)
- **Пустой ответ** не портит статистику и не закрывает вопрос
- **Мягкая проверка** развёрнутых ответов: пересечение ключевых слов с эталоном (4+ слова)
- Подсказки в `/help` и `/stats` про `/reset confirm` и `/review`

## 0.4.4 (2026-05-14)

- **`/review`** — повтор вопросов, где были ошибки
- **`/explain`** — пояснение к активному вопросу (без завершения сессии)
- **`/quiz 2`**, **`/quiz hard`**, **`/quiz ml-metrics 3`** — фильтр по сложности
- **`/reset confirm`** — подтверждение перед сбросом прогресса
- **+3 вопроса** (24 в банке)

## 0.4.3 (2026-05-14)

- **Меньше повторов**: приоритет вопросам, которые вы ещё не встречали (`question_history`)
- **Лучшая серия** (`best_streak`) в статистике и `/status`
- **`/progress`** — алиас `/stats` с прогрессом по банку (встречено X/Y)
- Сообщение при полном прохождении банка; пометка «Повтор вопроса»
- **+3 вопроса** (21 в банке)

## 0.4.2 (2026-05-14)

- **`/practice`** (и `/weak`) — вопрос по самой слабой или ещё не пройденной теме
- **`/topics`** — компактный список id тем и размер банка
- **Пояснения** (`explanation`) в банке — показываются после неверного ответа
- **`/map`**: число вопросов в банке по теме, id для `/quiz`
- **Меню команд** Telegram (`setMyCommands` при старте)
- **+3 вопроса** (18 в банке)

## 0.4.1 (2026-05-14)

- **`/hint`** — подсказка к активному вопросу без завершения сессии
- **Расширенная `/stats`**: серия верных, прогресс по темам, рекомендация что подтянуть
- **Серия ответов** (streak) в SQLite; бонусы на 3 / 5 / 10 подряд
- **+4 вопроса** в банке (15 всего); подсказки для статистики
- **`mentor check`**: проверка, что у каждой компетенции есть хотя бы один вопрос
